== Description ==

Adds links to share posts via Email, StumbleUpon, Digg, Facebook, Twitter. It's a really small, simple, efficient plugin which doesn't use javascript or flash.

Images for the links are included.

== Installation ==

1. Unzip the zip file and upload the "jw-share-this" folder into your Wordpress server's "plugins" directory.
2. Activate it from the plugins area of your admin pages.

== Screenshots ==

1. Share this section added to a post.

== Frequently Asked Questions ==

How do I use the plugin?

Install it and all posts will have a "share this" section added for visitors to share via Email, StumbleUpon, Digg, Facebook or Twitter.

== Uninstallation ==

Just deactivate the plugin and remove the files and folder. No data is stored anywhere else and no changes are made to any other files.
